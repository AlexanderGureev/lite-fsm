# Обзор

В этом разделе мы рассмотрим более сложные аспекты работы с `lite-fsm` и паттерны, которые можно использовать для решения нестандартных задач.

## Обзор разделов

- [Создание автомата](/guide/advanced-concepts/create-machine) - разбиение сложных автоматов на логические части для лучшей организации кода.
- [Вложенные/параллельные автоматы](/guide/advanced-concepts/parallel) - работа с несколькими автоматами, которые взаимодействуют друг с другом.
- [Отмена эффектов](/guide/advanced-concepts/cancel-effects) - встроенные механизмы для отмены эффектов при смене состояний.
- [Ручная отмена эффектов](/guide/advanced-concepts/manual-cancel) - программная отмена эффектов по условию.
- [Глобальные переходы](/guide/advanced-concepts/global-transitions) - переходы, которые применяются ко всем состояниям автомата.
- [Глобальные эффекты](/guide/advanced-concepts/global-effects) - эффекты, которые срабатывают при любых состояниях или переходах.
- [Debounced эффекты](/guide/advanced-concepts/debounced-effects) - отложенное выполнение эффектов с устранением лишних вызовов.
- [Обработка событий без перехода](/guide/advanced-concepts/null-transitions) - обработка действий, которые не должны менять состояние.
- [Условные переходы и guards](/guide/advanced-concepts/guards) - определение переходов, которые выполняются только при соблюдении определенных условий.
