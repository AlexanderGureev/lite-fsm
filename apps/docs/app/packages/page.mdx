# Пакеты lite-fsm

`lite-fsm` распространяется как набор npm-пакетов. `@lite-fsm/core` содержит независимый runtime для описания и запуска автоматов. Остальные пакеты добавляют React-интеграцию, middleware, persist-сценарии, анализ графа и CLI.

Базовый пакет — `@lite-fsm/core`. Остальные пакеты подключаются по необходимости, чтобы не добавлять в приложение лишние зависимости.

## Карта пакетов

| Пакет                                          | Роль                      | Когда нужен                                                      |
| ---------------------------------------------- | ------------------------- | ---------------------------------------------------------------- |
| [`@lite-fsm/core`](/packages/core)             | Runtime без привязки к UI | Автоматы, `MachineManager`, эффекты, акторы, snapshots           |
| [`@lite-fsm/react`](/packages/react)           | Интеграция с React        | Context provider, хуки, hydration boundary                       |
| [`@lite-fsm/middleware`](/packages/middleware) | Middleware для manager-а  | Immer reducers, Redux DevTools, пользовательские middleware      |
| [`@lite-fsm/persist`](/packages/persist)       | Сохранение состояния      | Storage adapters, восстановление, синхронизация между вкладками  |
| [`@lite-fsm/graph`](/packages/graph)           | Анализ графа автоматов    | Alpha: graph IR, diagnostics, simulator, модель для visualizer-а |
| [`@lite-fsm/cli`](/packages/cli)               | Командная строка          | Alpha: создание starter-а, add-machine, graph export и Visualizer |

## Установка по сценариям

Минимальный runtime:

```bash copy
npm install @lite-fsm/core
```

Runtime с Immer reducer-ами:

```bash copy
npm install @lite-fsm/core @lite-fsm/middleware immer
```

React-приложение:

```bash copy
npm install @lite-fsm/core @lite-fsm/react @lite-fsm/middleware immer
```

React + сохранение состояния:

```bash copy
npm install @lite-fsm/core @lite-fsm/react @lite-fsm/persist
```

CLI для старта React-проекта, добавления автоматов, анализа и визуализации графа:

```bash copy
npm install --save-dev @lite-fsm/cli
```

## Выбор пакетов

Начинайте с [`@lite-fsm/core`](/packages/core): он содержит модель автомата, `MachineManager`, эффекты, акторы и snapshot API. Подключайте [`@lite-fsm/react`](/packages/react), когда состояние нужно читать из React-компонентов через provider и хуки.

[`@lite-fsm/middleware`](/packages/middleware) нужен для reducer-ов в стиле Immer, Redux DevTools и собственных middleware. [`@lite-fsm/persist`](/packages/persist) используйте, когда состояние нужно сохранять между перезагрузками, восстанавливать из внешнего хранилища или синхронизировать между вкладками.

[`@lite-fsm/graph`](/packages/graph) предназначен для анализа и визуализации. [`@lite-fsm/cli`](/packages/cli) добавляет терминальные сценарии: создать starter, добавить автомат в generated store, экспортировать graph JSON или открыть локальный Visualizer. Эти пакеты находятся в alpha-версии. Graph-команды строят статическое представление автоматов и не выполняют пользовательские reducer-ы или effects.
