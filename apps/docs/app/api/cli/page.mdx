# CLI

`@lite-fsm/cli` публикует binary `lite-fsm`. Пакет находится в alpha-версии.

## `lite-fsm export-graph`

```bash copy
lite-fsm export-graph --entry store/index.ts --out lite-fsm.graph.json
```

Команда строит project graph через `@lite-fsm/graph` и пишет JSON export document.

| Опция | Обязательна | Назначение |
| --- | --- | --- |
| `--entry <path>` | Да | TypeScript entrypoint с выбранным `MachineManager(...)` |
| `--out <path>` | Да | Output JSON file |
| `--tsconfig <path>` | Нет | Явный tsconfig для module resolution |
| `--include-source` | Нет | Добавить source text в `sources.files[]` |

`--out -` не поддерживается: команда пишет только в файл.

## Diagnostics

CLI diagnostics используют коды `LFC_*`. Ошибки graph compiler остаются внутри `graph.diagnostics`, чтобы потребители export document могли отдельно показывать проблемы запуска CLI и проблемы анализа source.
