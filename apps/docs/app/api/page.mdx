## Быстрая карта

| Нужно                                                     | Куда идти                         |
| --------------------------------------------------------- | --------------------------------- |
| Описать автомат и запустить manager                       | [Core API](/api/core)             |
| Подключить состояние к React                              | [React API](/api/react)           |
| Мутировать reducer draft или подключить Redux DevTools    | [Middleware API](/api/middleware) |
| Сохранить snapshot в storage                              | [Persist API](/api/persist)       |
| Построить graph IR или diagnostics                        | [Graph API](/api/graph)           |
| Создать starter, добавить автомат или экспортировать graph | [CLI](/api/cli)                   |

## Типы

Основные типы живут рядом с публичными точками входа:

- `@lite-fsm/core`: events, configs, reducers, effects, managers, actors, snapshots, middleware.
- `@lite-fsm/react`: props и typed hook aliases.
- `@lite-fsm/persist`: storage adapters, controller, status и options.
- `@lite-fsm/graph`: IR, diagnostics, project host, simulator и view-model types.

Для сжатого справочника по типам используйте [`TYPES-CHEATSHEET.md`](https://github.com/AlexanderGureev/lite-fsm/blob/main/TYPES-CHEATSHEET.md).
