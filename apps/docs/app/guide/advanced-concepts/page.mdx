# Обзор

В этом разделе собраны паттерны для сценариев, где базового описания `config` + `reducer` уже недостаточно.

## Обзор разделов

- [Создание автомата](/guide/advanced-concepts/create-machine) — разбиение сложных автоматов на логические части.
- [Вложенные/параллельные автоматы](/guide/advanced-concepts/parallel) — работа с несколькими автоматами, которые взаимодействуют друг с другом.
- [Отмена эффектов](/guide/advanced-concepts/cancel-effects) — игнорирование transition-вызовов из устаревших запусков effects.
- [Ручная отмена эффектов](/guide/advanced-concepts/manual-cancel) — программная блокировка transition-вызовов по условию.
- [Глобальные переходы](/guide/advanced-concepts/global-transitions) — переходы, которые применяются ко всем состояниям автомата.
- [Глобальные эффекты](/guide/advanced-concepts/global-effects) — резервные effects для обработанных событий.
- [Debounced effects](/guide/advanced-concepts/debounced-effects) — отложенное выполнение effects с устранением лишних вызовов.
- [Обработка событий без перехода](/guide/advanced-concepts/self-transitions) — обработка actions, которые не должны менять состояние.
- [Условные переходы и guards](/guide/advanced-concepts/guards) — переходы, которые reducer может отклонить по условию.
